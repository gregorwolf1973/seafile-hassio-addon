## Add-ons

### Seafile

Self-hosted file sync and sharing platform. Runs as a standalone system inside Home Assistant OS.

**Features:**
- File sync across devices (Windows, Mac, Linux, Android, iOS)
- Web-based file manager (Seahub)
- File sharing with links and passwords
- Built-in MariaDB (no external database required)
- Persistent data on HA `/share` and `/config`

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the three-dot menu → **Repositories**
3. Add: `https://github.com/gregorwolf1973/seafile-hassio-addon`
4. Find **Seafile** and install it
